# Установка и запуск проекта

## 1. Установка зависимостей

### `yarn install` || `npm i`

### node.js - v16.16.0

### npm - 8.14.0

### yarn - 1.22.10

## 2. Установка json-server и json-server-auth глобально

### `yarn add -g json-server`

### `yarn add -g json-server-auth`

## 3. Запуск сервера

### `json-server-auth db.json -r route.json --port 3003`

## 4. Запуск проекта

### `yarn start`
